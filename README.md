# ⚡ Resume Engine — محرك السيرة الذاتية

نظام ذكي يحوّل أي نص خام إلى سيرة ذاتية احترافية في ثوانٍ.

---

## 🚀 تشغيل المشروع

### المتطلبات
- Python 3.9+  →  python.org
- Node.js 18+  →  nodejs.org
- Gemini API Key  →  aistudio.google.com

### على Windows
```
انقر مرتين على: start.bat
```

### على Mac/Linux
```bash
pip install -r requirements.txt
npm install -g docx
python app.py
```

ثم افتح المتصفح على: **http://localhost:5000**

---

## 🔑 إضافة API Key
1. افتح التطبيق في المتصفح
2. في خانة "مفتاح Gemini API" الصف العلوي
3. الصق الـ Key واضغط "حفظ"
4. يُحفظ تلقائياً في المتصفح

---

## 📋 طريقة الاستخدام
1. **الصق النص الخام** — أي نص يحتوي بياناتك (حتى لو مش منظم)
2. **اختر القالب** — إنجليزي مودرن أو عربي كلاسيك
3. **اضغط "توليد"** — يتحمل ملف Word جاهز خلال 30 ثانية

---

## 📁 هيكل المشروع
```
resume-engine/
├── app.py              ← السيرفر الرئيسي
├── requirements.txt    ← متطلبات Python
├── start.bat           ← ملف التشغيل (Windows)
├── static/
│   └── index.html      ← الواجهة الأمامية
└── templates/
    ├── en_modern.js    ← قالب إنجليزي مودرن
    └── ar_classic.js   ← قالب عربي كلاسيك
```

---

## ➕ إضافة قالب جديد
1. انسخ `templates/en_modern.js`
2. غير الألوان والتنسيق
3. احفظ باسم جديد مثل `en_executive.js`
4. أضف الخيار في `static/index.html`

---

## 🔒 الأمان والخصوصية
- النظام يشتغل **محلياً** على جهازك
- البيانات **لا ترسل** لأي موقع خارجي (عدا Gemini API للتحليل فقط)
- الـ API Key يُحفظ في المتصفح فقط ولا يُخزن على أي سيرفر
